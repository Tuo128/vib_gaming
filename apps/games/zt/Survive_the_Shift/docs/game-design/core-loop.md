# 核心循环

## 一局的时间结构

| 游戏时间 | 阶段 | 体验目标 |
| --- | --- | --- |
| 9:00–10:30 | 热身 | 学会移动与第一件武器 |
| 10:30 | 需求评审 | 第一次机制检验 |
| 10:30–14:00 | 成长 | 形成武器组合，敌潮增密 |
| 14:00 | 无效会议 | 限制走位和持续施压 |
| 14:00–16:30 | 爆发 | 构筑进入成型期 |
| 16:30 | 需求变更 | 改变敌群组合 |
| 16:30–17:59 | 顶峰 | 大量清怪和精英混编 |
| 17:59 | 最后一个需求 | 撤离前 Boss |
| 18:00 之后 | 下班撤离 | 抵达电梯完成本局 |

## 每分钟循环

1. 观察敌群和安全空间。
2. 移动以维持武器输出。
3. 清除高威胁或高待办敌人。
4. 收集经验和恢复资源。
5. 升级并调整构筑方向。
6. 决定是否进入摸鱼区。

## 胜利

达到 18:00 后，地图生成电梯出口。玩家进入电梯范围并完成短暂开门过程后获胜。

## 失败

- 心态归零：精神崩溃。
- 待办达到上限：无限加班。

## 节奏要求

- 开局 20 秒内出现第一批敌人。
- 60–90 秒内第一次升级。
- 当前 3 分钟测试局每 30–45 秒出现新的敌人组合或事件。
- 构筑成型后允许玩家短时间获得压倒性优势。
- 最后一分钟重新制造压力，但不能完全否定已形成的构筑。
