## 玩法支柱

1. 同屏大量敌人的快速清除。
2. 每分钟都能感受到构筑成长。
3. 心态与待办形成双重压力。
4. 摸鱼提供高收益但会制造新风险。
5. 18:00 后必须完成撤离。
